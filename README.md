# 俄语高考词汇学习应用

一个专为俄语高考备考设计的词汇学习工具，支持间隔重复学习、多种测验模式，数据本地存储。

## 功能特性

### 词库浏览
- 收录高考核心词汇约 2000 个
- 支持搜索和筛选（按掌握程度分类）
- 卡片式展示，左侧边框颜色标识学习状态

### 学习模式
- 基于 SM-2 算法的间隔重复系统
- 卡片翻转学习，点击查看释义
- 三级评分：忘记 / 困难 / 简单
- 自动安排复习计划

### 测验模式
- **选择题**：看俄语选中文释义
- **拼写题**：看中文拼俄语单词（字母池点击输入）
- 实时反馈正确/错误

### 记忆游戏
- 配对游戏：将俄语单词与中文释义配对
- 计时挑战

### 进度追踪
- 学习统计：已学数量、掌握程度
- 本地存储，无需账号
- 支持重置进度

## 技术栈

- 纯前端实现：HTML + CSS + JavaScript
- 无框架依赖，无需构建
- 数据存储：localStorage
- 响应式设计，适配移动端

## 使用方法

直接用浏览器打开 `index.html` 即可使用。

```bash
# 或使用本地服务器
python -m http.server 8080
# 访问 http://localhost:8080
```

## 文件结构

```
public/
├── index.html      # 主应用文件
├── words_data.js   # 词汇数据库
└── README.md       # 项目说明
```

## 数据格式

词汇数据格式：

```javascript
const words = [
  { "word": "авария", "meaning": "事故；故障" },
  { "word": "август", "meaning": "八月" },
  // ...
];
```

学习进度存储结构：

```javascript
{
  "авария": {
    "level": 2,           // 掌握等级 0-5
    "nextReview": 1714387200000,  // 下次复习时间戳
    "lastReview": 1714300800000,
    "reviewCount": 3
  }
}
```

## 设计风格

- 复古纸张风格（米黄色调）
- 衬线字体：Playfair Display（俄语）、Noto Serif SC（中文）
- 底部导航栏，移动端友好

## 自定义词汇

编辑 `words_data.js` 文件，按相同格式添加或修改词汇即可。
